# ⚡ VibeDiff (バイブ・ディフ)

**AI時代のバイブコーダーに、「安心」と「理解」を。**

VibeDiffは、CursorやCline、ChatGPTといったAIエージェントと共に開発を進める**「バイブコーディング」**に最適化された、ローカル完結型の次世代差分解析ツールです。

---

## 🧐 なぜ VibeDiff が必要なのか？

AIを使った開発は驚くほど速いですが、同時に**「何が起きたか分からない」**という不安も生みます。

- **ブラックボックス化**: AIが数百行のコードを一瞬で書き換えたとき、その本質的な意図を掴むのは困難です。
- **Gitの限界**: 従来のDiffは「行の変化」を教えるだけです。非エンジニアや初心者にとって、それはただの記号の羅列に見えてしまいます。
- **デプロイの恐怖**: 「AIを信じてデプロイしたけど、実は重要なファイルを壊していた」という事故を防ぐには、意味レベルでの確認が不可欠です。

**VibeDiffは、コードの「記号」を「言葉」に翻訳し、あなたのバイブス（直感）を確信に変えます。**

---

## ✨ 圧倒的な 4 つの特徴

### 1. 「役割」にフォーカスした論理翻訳
単なる差分表示ではありません。「このファイルはUIの心臓部です」「この変更は、外部データベースとの通信を司っています」といったように、アプリ内の**役割に基づいた日本語解説**を生成します。

### 2. フォルダごと一撃で解析
1ファイルずつ比較する必要はありません。プロジェクトフォルダを選択するだけで、追加・削除・変更されたすべてのファイルを網羅。`node_modules` などの巨大な不要フォルダは賢くスキップし、人間がチェックすべき本質だけを抽出します。

### 3. 多言語・多環境への高度な知能
- **Web**: React, Next.js, Vite, Supabase 等
- **Mobile**: iOS (Swift), SwiftUI
- **Automation**: Python, Batch scripts
- **Game**: WalkCraft等、特殊な素材フォルダ構造

ファイル名や内容からプロジェクトを自動判定し、最適な文脈で解説します。

### 4. 究極のプライバシー（Local-First）
**あなたのソースコードは、一文字もインターネットへ送信されません。**
最新のブラウザ技術（File System Access API）を活用し、すべての解析処理はあなたのPC上のブラウザ内だけで完結します。機密性の高い商用プロジェクトや個人の秘伝のタレも、安心して読み込ませてください。

---

## 🛠 技術的な裏側

VibeDiff自体も、最先端の「サーバーレス・シングルページ・アーキテクチャ」で構築されています。

- **コア**: [React 18](https://react.dev/) + Hooks
- **解析エンジン**: 独自の正規表現パターンマッチングによる「コードの意味抽出」
- **コンパイラ**: [Babel standalone](https://babeljs.io/)（ブラウザ上でReactを即時実行）
- **高速DIFF**: [jsdiff](https://github.com/kpdecker/jsdiff) による行単位・文字単位の比較
- **設計**: HTMLファイル一つで完結するため、環境構築不要でどこでも動作します。

---

## 🚀 使い方

1. [VibeDiff 公式ページ](https://oyajibuki.github.io/diff/) を開く。
2. **Step 1 (分析)**: 修正前のフォルダを選択。
3. **Step 2 (比較)**: 修正後のフォルダを選択。
4. **理解する**: 左側のツリーからファイル（またはスキップされたフォルダ）を選択し、右側の「役割と動き」を確認します。
